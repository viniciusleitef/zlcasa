# zlcasa - Crafted Porcelain & Ceramics

Peças exclusivas de porcelana e cerâmica pintadas à mão. Lembranças para batizados, casamentos e eventos especiais. Arte que transforma momentos em memórias.

## Tecnologias

Este projeto foi desenvolvido utilizando:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Desenvolvimento

Para rodar o projeto localmente:

1. Instale as dependências:
   ```sh
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```sh
   npm run dev
   ```

## Build

Para gerar a versão de produção:
```sh
npm run build
```
